Доброго времени суток!
Могу сразу предупредить, что из необходимого функционала приложения не реализована функция получения информация о состоянии памяти устройства, остальные функции должны работать корректно.
Для реализации приложения использовалась библиотека Managed Android Debug Bridge (Mad Bee).
Для корректной работы приложения в конфигурационном файле необходимо в переменной AbdPath указать корректный путь к файлу adb.exe. Также на самом устройстве должна быть включена функция отладки через USB.
